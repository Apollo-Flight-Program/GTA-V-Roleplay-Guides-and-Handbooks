# Charges and Sentencing Reference

A reference guide to common charges filed in GTA V roleplay and a sentencing framework. Use this alongside a server's Penal Code to understand what charges apply to what behavior and what consequences are proportional.

---

## How Charges Work in RP

### Felony vs. Misdemeanor
| Class | Description | Typical Sentence |
|---|---|---|
| **Felony 1 (F1)** | Most serious violent crimes | 10+ years or life |
| **Felony 2 (F2)** | Serious violent/property crimes | 5–10 years |
| **Felony 3 (F3)** | Lesser felonies | 2–5 years |
| **Misdemeanor 1 (M1)** | Serious misdemeanor | Up to 1 year jail, fine |
| **Misdemeanor 2 (M2)** | Moderate misdemeanor | Fine, community service |
| **Infraction** | Minor violation | Fine only |

*Server penal codes vary. Use this as a framework, not an absolute.*

## Sentencing Considerations

### Aggravating Factors (Increase Sentence)
- Prior criminal record
- Use of a weapon
- Targeting vulnerable victims (elderly, children)
- Commission of the offense in concert with others
- Position of trust (police officer, employer)
- Particularly cruel or heinous conduct

### Mitigating Factors (Decrease Sentence)
- First offense; no prior record
- Cooperation with law enforcement
- Genuine remorse
- Minor role in the offense
- Diminished mental capacity
- Restitution voluntarily made

### Sentence Types
| Type | Description |
|---|---|
| **Imprisonment** | Incarceration in city jail (short) or state prison (long sentences) |
| **Probation** | Supervised release; must meet with a probation officer and comply with conditions |
| **Fines** | Monetary penalty |
| **Community service** | Assigned hours of unpaid work benefiting the community |
| **Restitution** | Payment to the victim for harm caused |
| **License suspension** | Loss of driving or professional license |
| **Concurrent** | Multiple sentences run at the same time |
| **Consecutive** | Multiple sentences run back-to-back (often harsher) |
