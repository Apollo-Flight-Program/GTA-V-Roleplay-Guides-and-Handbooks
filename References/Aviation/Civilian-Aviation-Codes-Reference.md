# Civilian Aviation Codes Reference

A reference guide for civilian aviation roleplay covering VFR/IFR rules, transponder codes, ATIS broadcasts, flight planning, and civil aviation authority interactions.

---

## Visual Flight Rules (VFR) vs Instrument Flight Rules (IFR)

| Rule Set | Conditions | Pilot Requirement |
|---|---|---|
| **VFR** | Clear skies; can see to navigate visually | Basic pilot certificate; no instrument rating required |
| **IFR** | Low visibility, clouds, night (some cases) | Instrument rating required; flight plan filed with ATC |
| **MVFR** | Marginal VFR; technically legal but degraded visibility | Caution; near the edge of visual conditions |
| **IMC** | Instrument Meteorological Conditions; IFR required | Must file and fly IFR |

## Transponder Codes

The transponder squawks a 4-digit code that identifies the aircraft to radar.

| Code | Meaning |
|---|---|
| **1200** | Default VFR code in uncontrolled airspace |
| **7500** | **Hijacking / unlawful interference** — immediately alerts ATC and authorities |
| **7600** | Radio failure — aircraft is unable to communicate |
| **7700** | **Emergency** — general distress; highest priority alert |
| **Assigned code** | ATC assigns a unique code when entering controlled airspace |

---

## ATIS (Automatic Terminal Information Service)

Recorded voice broadcast on a dedicated frequency at controlled airports. Provides:
- Weather conditions (wind, visibility, cloud ceiling, temperature, dewpoint)
- Active runways
- NOTAMs (Notices to Airmen — special procedures in effect)
- Information identifier (letter — "Information Bravo")

**Standard acknowledgement:** *"[Airport] Tower, [Callsign], have information [letter]."*

## Flight Plan Filing

Filed before IFR flights and recommended for VFR cross-country.

| Field | Content |
|---|---|
| Aircraft type | Make, model, and registration |
| Departure | ICAO code of departure airport |
| Destination | ICAO code of destination airport |
| Route | Fixes, airways, or direct routing |
| Altitude | Planned cruise altitude |
| Estimated time en route | Total flight time |
| Fuel on board | Time of fuel available |
| Alternate | Alternate airport if destination is unavailable |

## ATC Communication — Civilian Standard

### Callsign Format
**[Aircraft type or airline] + [Registration/Flight number]**
- Light aircraft: *"Cessna 1-7-Sierra-Foxtrot"*
- Business jet: *"Citation 4-4-Echo-Kilo"*
- Charter flight: *"Los Santos Air 347"*

### Standard Contact Sequence

1. State the facility you are calling
2. Your callsign
3. Your position and altitude
4. What you want

*"Los Santos Approach, Cessna 1-7-Sierra-Foxtrot, 10 miles south, 3,500 feet, request VFR flight following to Paleto Bay."*

## Civilian Airspace Classes

| Class | Type | Who Can Enter |
|---|---|---|
| **Class A** | High-altitude en route | IFR only; above 18,000 feet |
| **Class B** | Major airport area | Clearance required; high traffic |
| **Class C** | Regional airport area | Clearance required; radar service |
| **Class D** | Smaller controlled airports | Two-way radio contact required |
| **Class E** | Controlled airspace, no tower | IFR clearance required in some conditions |
| **Class G** | Uncontrolled | No ATC requirement; pilot's own judgment |

## Pilot Certificates and Privileges

| Certificate | What it Allows |
|---|---|
| **Student Pilot** | Solo flight only in certain aircraft; restricted |
| **Private Pilot (PPL)** | Fly for pleasure or personal business; no compensation |
| **Commercial Pilot (CPL)** | Fly for hire; charter, banner towing, flight instruction |
| **Airline Transport Pilot (ATP)** | Required to serve as captain on airline-size aircraft |
| **Instrument Rating (IR)** | Fly in IFR conditions; filed flight plans |
| **Multi-Engine Rating (ME)** | Fly aircraft with more than one engine |

## Emergency Procedures

### In-Flight Emergency
1. **Aviate** — Fly the aircraft first
2. **Navigate** — Know where you are and where you can land
3. **Communicate** — Declare emergency on 121.5 (guard) or current frequency

### Emergency Declaration
*"Mayday, Mayday, Mayday — [callsign], [nature of emergency], [position], [altitude], [souls on board], [fuel state]."*

Or for urgent but non-distress:
*"Pan-Pan, Pan-Pan, Pan-Pan — [callsign], [situation]."*

### Emergency Landing Options
1. Designated emergency airport
2. Highway / straight road (last resort)
3. Open field
4. Aim for the least populated, most accessible area

## Common Aviation Phonetics (Aviation-Specific)

| Term | Meaning |
|---|---|
| **Affirm / Negative** | Yes / No (not "affirmative" in full) |
| **Wilco** | Will comply |
| **Unable** | Cannot comply with clearance |
| **Say altitude** | Confirm current altitude |
| **Ident** | Press the transponder ident button to help ATC identify your return |
| **Squawk VFR** | Return to code 1200 |
| **Traffic in sight** | Visual contact with another aircraft ATC called |
| **Report field in sight** | Tell ATC when you can see the airport |
