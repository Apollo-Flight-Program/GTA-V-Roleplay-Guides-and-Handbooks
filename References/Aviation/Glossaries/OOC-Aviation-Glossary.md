# Aviation — Out-of-Character Glossary

Terms used by **players** when discussing aviation roleplay outside of character. Covers the mechanics, player conventions, and community concepts for civilian and military aviation on an RP server.

---

## Roles and Whitelisting

| Term | OOC Meaning |
|---|---|
| **Aviation whitelist** | A role requiring an approved application, usually including a flight test in-game, to operate certain aircraft. |
| **Pilot license** | An IC document and OOC permission tier that determines what aircraft a character is permitted to fly. |
| **PPL (Private Pilot License)** | Entry-level license — single-engine prop aircraft, VFR only. |
| **CPL (Commercial Pilot License)** | Intermediate license — may carry passengers and operate commercially. |
| **ATPL (Airline Transport Pilot License)** | Highest civilian license tier — multi-engine, jets, instrument rated. |
| **Flight test / Check ride** | The in-game evaluation where a player demonstrates flying ability to earn a license tier. Administered by a qualified instructor character. |
| **CFI (Certified Flight Instructor)** | A character authorized to conduct flight training and administer check rides. |
| **Police aviation** | Law enforcement pilots — usually a LSPD/BCSO Air Support Unit. Separate from civilian aviation. |
| **Military aviation** | SAMAF pilots and crew — separate whitelist from civilian aviation. See [Military Aviation Terms](../Military-Aviation-Terms-and-Jargon.md). |

## Aircraft and Server Mechanics

| Term | OOC Meaning |
|---|---|
| **Spawn aircraft / Spawn heli** | The command or menu option to bring an aircraft into the game world. Usually limited by whitelist tier and vehicle permissions. |
| **Aircraft script** | Server-side code that modifies how aircraft behave — fuel consumption, damage modeling, radio integration, etc. |
| **Fuel script** | An add-on that gives aircraft realistic fuel that must be refueled at airports or helipads. |
| **Radio script / VCRP** | In-game voice or text radio that simulates ATC and pilot communication on specific frequencies. |
| **Transponder script** | In-game mechanic that broadcasts aircraft position, altitude, and squawk code to ATC. |
| **Crash protection** | Some servers reduce or eliminate crash damage for certain scripted aircraft scenarios to allow cleaner RP. |
| **Restricted airspace** | A server-defined zone where aircraft are prohibited from flying without clearance — military zones, events, etc. |
| **LSIA** | Los Santos International Airport — the primary civilian aviation hub. |
| **Sandy Shores Airfield** | Secondary airfield in Blaine County. Used for light aviation, often criminal logistics RP. |
| **McKenzie Airfield** | A smaller airstrip used for prop aircraft and bush flying RP. |
| **Fort Zancudo** | The military base in northern Blaine County. Military aviation only — entering without authorization is a server rule violation. |

## Flight RP Conventions

| Term | OOC Meaning |
|---|---|
| **IFR filing** | Filing a written flight plan in a CAD or server aviation system before departure. Required on some servers for all flights. |
| **Flight following** | A light version of IFR — ATC tracks the aircraft and provides advisories without full control. |
| **Uncontrolled airspace** | Areas where no ATC service is available — pilots are responsible for see-and-avoid. |
| **Pattern work** | Flying repeated circuits around an airport to practice takeoffs and landings. |
| **Touch and go RP** | Training RP where a character practices multiple landings in succession. |
| **Air support RP** | Aviation providing assistance to a ground unit — surveillance, transportation, armed overwatch (military). |
| **HEMS (Helicopter Emergency Medical Services)** | Air ambulance RP — EMS and pilot coordinating for aerial patient transport. |
| **HOIST RP** | A complex rescue scenario involving lowering a person or equipment from a hover. Requires scripted support or heavy /me use. |
| **Interception RP** | Military aviation intercepting an unauthorized aircraft — a multi-unit scenario involving both pilots. |
| **Crash RP** | Roleplaying a crash or emergency landing. Requires consent from all involved players. |
