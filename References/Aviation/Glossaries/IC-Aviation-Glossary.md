# Aviation — In-Character Glossary

Language used by **aviation characters** in the world of Los Santos — civilian pilots, air traffic control, police aviation, and military aviators. These are the words and phrases used in cockpits, on radio frequencies, and in the aviation community.

For military-specific aviation terminology, see [Military Aviation Terms and Jargon](../Military-Aviation-Terms-and-Jargon.md).

---

## Pilot Communication Language

### Standard Radio Phrases

| Term | IC Meaning |
|---|---|
| **"With you"** | Acknowledging on a new frequency or checking in with a new controller. "Los Santos Approach, Cessna N-4-4-Lima, with you at five thousand." |
| **"Negative contact"** | Cannot see or detect the referenced traffic or target. |
| **"Traffic in sight"** | The aircraft or hazard mentioned by ATC is visually identified. |
| **"Squawk"** | Set the transponder to a specific code. "Squawk 4523." |
| **"Ident"** | Press the transponder ident button so the controller can identify your blip on radar. |
| **"Cleared"** | Authorized to proceed with the stated action. "Cleared for takeoff, Runway 21." |
| **"Wilco"** | Will comply — received the instruction and will follow it. |
| **"Unable"** | Cannot comply with the instruction as given. Must state the reason. |
| **"Say altitude"** | Request the current altitude of the aircraft. |
| **"Say airspeed"** | Request the current airspeed of the aircraft. |
| **"Say intentions"** | Request the pilot's plan — where are they going, what are they doing. |
| **"Stand by"** | Wait — the controller is busy or will respond shortly. |
| **"Go ahead"** | Ready to receive the transmission. |
| **"Read back"** | Repeat the clearance or instruction back to confirm receipt. |
| **"Correction"** | Indicates the previous transmission contained an error. |

### Position and Navigation

| Term | IC Meaning |
|---|---|
| **Heading** | The direction the aircraft is pointed, in degrees magnetic. "Turn to heading 270." |
| **Track** | The actual path of the aircraft over the ground (may differ from heading due to wind). |
| **Altitude** | Height above mean sea level, in feet. |
| **Flight level (FL)** | Altitude expressed in hundreds of feet above the transition altitude. FL150 = 15,000 feet. |
| **Angels** | Military/aviation slang for altitude in thousands of feet. "Angels 10" = 10,000 feet. |
| **Waypoint** | A defined geographic position used in navigation. |
| **Fix** | A confirmed position. "I have a position fix." |
| **Bearing** | Direction from one point to another, in degrees. |
| **Radial** | A line extending outward from a navigation beacon at a specific bearing. |
| **VFR (Visual Flight Rules)** | Flying with visual reference to the ground. Requires clear visibility. |
| **IFR (Instrument Flight Rules)** | Flying using instruments only — in clouds or low visibility. |
| **VMC / IMC** | Visual Meteorological Conditions / Instrument Meteorological Conditions. |
| **On course** | Flying in the correct direction toward the destination. |
| **Deviation** | Flying off the planned route. |

### Takeoff and Landing

| Term | IC Meaning |
|---|---|
| **Rotate** | The moment the aircraft lifts off the runway. "V-R, rotate." |
| **V speeds** | Critical aircraft speed milestones. V1 = takeoff decision speed, Vr = rotation speed, V2 = takeoff safety speed. |
| **Final approach / Final** | The last segment of the approach, aligned with the runway, descending to land. |
| **Short final** | Very close to the runway on final approach — landing is imminent. |
| **Flare** | The pitch-up maneuver just before touchdown to slow descent and touch down smoothly. |
| **Touchdown** | The moment the wheels contact the runway. |
| **Rollout** | Deceleration after touchdown along the runway. |
| **Go-around** | Aborting the landing approach — adding power and climbing to try again. |
| **Balked landing** | Same as go-around — an aborted landing. |
| **Cleared to land** | ATC authorization to continue to a full-stop landing. |
| **Touch and go** | A landing immediately followed by a takeoff, without stopping. Used for practice. |

### Aircraft Status

| Term | IC Meaning |
|---|---|
| **Airborne** | In flight. "We are airborne, heading northwest." |
| **On deck / On the ground** | Aircraft has landed or has not departed. |
| **Established** | Settled into a pattern, holding altitude, or aligned with approach. |
| **Orbiting** | Flying in circles around a point, usually for surveillance or waiting. |
| **Holding** | A specific racetrack-pattern maneuver to delay progress while waiting for clearance. |
| **Bingo fuel** | Fuel state at the minimum required to return to base safely. "Bingo in twenty minutes." |
| **Fumes** | Dangerously low fuel — less than minimum. Emergency situation. |
| **Emergency declared** | Pilot has declared an in-flight emergency. ATC clears all resources to assist. |
| **Pan-pan** | Urgent but not immediate distress — a serious situation that may become an emergency. |
| **Mayday** | International distress call. Life is in danger. The highest priority radio call. |

## Civilian Aviation Slang

| Term | IC Meaning |
|---|---|
| **Bus driver** *(slang)* | A commercial airline pilot — sometimes slightly derogatory among GA (general aviation) pilots. |
| **Stick and rudder** | The basic physical art of flying an aircraft. "Old-school stick and rudder skills." |
| **Hangar flying** *(slang)* | Informal conversation among pilots about flying — telling stories, discussing aircraft. |
| **NOTAM** | Notice to Airmen — official notice about hazards, restrictions, or changes to airspace. |
| **Squall** | A sudden severe weather condition — lightning, heavy rain, strong wind. |
| **Greased it** *(slang)* | An exceptionally smooth landing. "He really greased that one on." |
| **Pilot in Command (PIC)** | The pilot legally responsible for the flight. Even if others are aboard, one person holds PIC authority. |
| **Right seat / Left seat** | In a two-person cockpit: left seat is PIC, right seat is co-pilot or passenger. |
