# Street Slang Reference

A reference guide to street-level language used in criminal, gang, and urban civilian roleplay. These terms are meant to be used **in character** — as natural vocabulary by characters who inhabit that world.

Terms are grouped by domain. Many terms have regional variations; this list reflects a broadly understood urban vernacular appropriate for a Los Santos setting.

---

## Money and Finance

| **Term** | Meaning |
|---|---|
| **Bread / Guap / Racks** | Money |
| **Band / Bandz** | $1,000. *"I need ten bands."* |
| **A stack** | $1,000 |
| **A brick** | $1,000 or a kilogram of a drug, depending on context |
| **Paper / Cheese / Cake** | General term for cash |
| **Drip** | Style and wealth projected through clothing and accessories |
| **Bag** | A financial opportunity or a large sum of money. *"Secure the bag."* |
| **Laundering / Washing** | Making illegal money appear legitimate |
| **The books** | Accounting records, sometimes falsified |
| **Clean money** | Legitimately earned or laundered funds |

## Law Enforcement and Heat

| **Term** | Meaning |
|---|---|
| **5-0 / Po-Po / Twelve** | Police |
| **The boys in blue** | Police officers |
| **Jakes** | Plainclothes or undercover police |
| **Fed / Feds** | Federal agents; FBI equivalent |
| **Heat** | Police attention. *"We got heat on us right now."* |
| **Hot** | Under police surveillance or suspicion |
| **Dirty** | Carrying contraband or weapons when stopped |
| **Clean** | Not carrying anything incriminating |
| **Booked** | Arrested and processed |
| **Caught a charge** | Was arrested and charged with a crime |
| **Case** | A criminal investigation or pending charge |
| **Snitching / Telling** | Giving information to the police about others |
| **Rat / Snitch** | Someone who informs on others to law enforcement |
| **Cooperating** | Working with law enforcement, often for a deal |
| **Take the charge** | Accept legal responsibility, often to protect others |
| **Walk** | Be released or acquitted. *"He walked on the charges."* |

## Drugs

| **Term** | Meaning |
|---|---|
| **Work** | Drugs for sale |
| **Product / Package** | A drug supply |
| **Soft** | Powder cocaine |
| **Hard / Crack** | Crack cocaine |
| **Loud / Gas** | High-quality marijuana |
| **Reggie** | Low-quality, cheap marijuana |
| **Ice / Crystal / Glass** | Methamphetamine |
| **Molly / Ecstasy** | MDMA |
| **Wock / Lean / Drank** | Prescription cough syrup mixed drink |
| **Trap / Trap house** | Location where drugs are sold |
| **Serve / Sling** | Sell drugs |
| **Plug** | Drug supplier |
| **Connect** | Source of supply |
| **Flip** | Resell. *"I'll flip this pack and re-up."* |
| **Re-up** | Restock supply |
| **Cut** | Adulterant mixed into drugs to increase weight |
| **A front** | Supply given on credit; payment due after sale |

## Violence and Conflict

| **Term** | Meaning |
|---|---|
| **Beef** | Ongoing conflict with another person or group |
| **Squash the beef** | Resolve a conflict |
| **Shoot on sight (SOS)** | An order to attack a person on contact |
| **Spin the block** | Return to an area to retaliate or cause violence |
| **Get pressed** | Be confronted aggressively |
| **Run up** | Approach aggressively or attack |
| **Slide** | Go to a location with hostile intent |
| **Fade** | A one-on-one physical fight |
| **Put in work** | Commit violence on behalf of a group |
| **Caught a body** | Killed someone |
| **Smoking / Touching** | Shooting or harming someone |
| **Blicky / Stick / Pole** | Firearm |
| **Draco** | AK pistol; also used generically for automatic pistol |
| **Extendo / Switch** | Extended magazine or auto-conversion device |
| **Strapped / Dirty** | Armed with a weapon |

## Gang and Group Terms

| **Term** | Meaning |
|---|---|
| **Set** | A specific gang clique or subdivision |
| **Crew** | A tight group of associates |
| **Block / Turf** | The physical territory controlled by a group |
| **Claim** | To represent or be affiliated with a group |
| **Rep** | Represent a gang or neighborhood. Also: reputation. |
| **Homie / Folks / Fam** | A close associate; member of the same group |
| **OG (Original Gangster)** | Senior, respected gang member |
| **Young'un / Shorty** | A newer, younger member |
| **Foot soldier** | Lower-ranking member who carries out tasks |
| **Shot caller** | Person who makes decisions for the group |
| **Plug in** | Connect a new member to the organization |
| **Put on** | Initiated someone into the group |
| **Violated** | Broke a code or rule of the organization; subject to punishment |
| **Flip** | Change gang affiliation |

## General Street Terms

| **Term** | Meaning |
|---|---|
| **On sight** | Will engage immediately upon seeing; no hesitation |
| **Real talk / On god** | I'm being completely serious |
| **Play yourself** | Act foolishly in a way that invites harm |
| **Catch an L** | Suffer a loss |
| **Move different** | Behave unusually; breaking expectations |
| **Ghost / Dip / Bounce** | Leave quickly |
| **Low key** | Quietly; without drawing attention |
| **Keep it a buck (100)** | Be completely honest |
| **Fronting** | Pretending to be something you're not |
| **Solid** | Trustworthy, reliable |
| **Snake** | Untrustworthy, treacherous person |
| **Cap / Capping** | Lying. *"No cap"* = I'm not lying. |
| **Sauce / Swag** | Style, confidence, charisma |
