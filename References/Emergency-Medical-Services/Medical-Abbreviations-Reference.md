# Medical Abbreviations Reference

A comprehensive reference of medical abbreviations and shorthand used in EMS roleplay — in written reports, radio communications, and scene documentation. These terms appear in patient care reports, hospital handoffs, and inter-agency communications.

---

## Patient Assessment Abbreviations

| Abbreviation | Meaning |
|---|---|
| **A&Ox4** | Alert and oriented to person, place, time, and event — fully oriented |
| **A&Ox3** | Alert and oriented to person, place, and time (not event) |
| **A&Ox2** | Alert and oriented to person and place only |
| **A&Ox1** | Alert and oriented to person only |
| **GCS** | Glasgow Coma Scale (3–15) |
| **LOC** | Level of consciousness |
| **LOC (alt.)** | Loss of consciousness |
| **Δ LOC** | Change in level of consciousness (Δ = change) |
| **PERRL** | Pupils equal, round, and reactive to light |
| **PERRLA** | Pupils equal, round, and reactive to light and accommodation |
| **Aniso** | Anisocoria — unequal pupil size |
| **ETOH** | Ethanol — alcohol intoxication |
| **SOB** | Shortness of breath |
| **DOE** | Dyspnea on exertion — shortness of breath with activity |
| **CP** | Chest pain |
| **N/V/D** | Nausea, vomiting, diarrhea |
| **Hx** | History |
| **Sx** | Symptoms |
| **Dx** | Diagnosis |
| **Tx** | Treatment |
| **Px** | Prognosis |
| **Rx** | Prescription / medication |
| **CC** | Chief complaint |
| **MOI** | Mechanism of injury |
| **NOI** | Nature of illness |
| **PMH** | Past medical history |
| **HPI** | History of present illness |
| **OPQRST** | Pain assessment: Onset, Provocation/Palliation, Quality, Radiation, Severity, Time |
| **SAMPLE** | Patient history: Signs/Symptoms, Allergies, Medications, Pertinent history, Last oral intake, Events preceding |

## Vital Signs

| Abbreviation | Meaning |
|---|---|
| **HR** | Heart rate (beats per minute) |
| **RR** | Respiratory rate (breaths per minute) |
| **BP** | Blood pressure (systolic/diastolic mmHg) |
| **SpO2 / O2 Sat** | Oxygen saturation (pulse oximetry, percentage) |
| **Temp** | Temperature |
| **BGL / BGS** | Blood glucose level / blood glucose stick |
| **Tachycardia** | Heart rate > 100 bpm |
| **Bradycardia** | Heart rate < 60 bpm |
| **Hypertension** | Blood pressure abnormally high |
| **Hypotension** | Blood pressure abnormally low |
| **Tachypnea** | Respiratory rate > 20 breaths/min |
| **Bradypnea** | Respiratory rate < 12 breaths/min |
| **Apnea** | Absence of breathing |
| **Diaphoretic** | Sweating profusely |

## Treatments and Interventions

| Abbreviation | Meaning |
|---|---|
| **IV** | Intravenous (into a vein) |
| **IO** | Intraosseous (into bone marrow — used when IV access unavailable) |
| **O2** | Oxygen |
| **NRB** | Non-rebreather mask — high-flow oxygen delivery |
| **NC** | Nasal cannula — low-flow oxygen |
| **BVM** | Bag-valve mask |
| **ET tube / ETT** | Endotracheal tube — advanced airway |
| **OPA** | Oropharyngeal airway |
| **NPA** | Nasopharyngeal airway |
| **TQ** | Tourniquet |
| **PPE** | Personal protective equipment |
| **NS** | Normal saline (IV fluid) |
| **LR** | Lactated Ringer's (IV fluid) |
| **CPR** | Cardiopulmonary resuscitation |
| **AED** | Automated external defibrillator |
| **NTG** | Nitroglycerin (chest pain medication) |
| **ASA** | Aspirin |
| **EPI / Epi** | Epinephrine |
| **NX / Narcan** | Naloxone — opioid reversal agent |
| **Glucose** | Dextrose — given for low blood sugar |
| **Intubated** | Advanced airway placed, patient being ventilated |
| **Cric** | Cricothyrotomy — surgical emergency airway |
| **NCD** | Needle chest decompression |
| **ROSC** | Return of spontaneous circulation |
| **DNR** | Do not resuscitate order |

## Documentation Shorthand

| Abbreviation | Meaning |
|---|---|
| **PCR** | Patient care report |
| **DOA** | Dead on arrival |
| **DOT** | Dead on transport |
| **c/o** | Complains of |
| **s/p** | Status post (after — e.g., "s/p MVA" = following a vehicle collision) |
| **R/O** | Rule out (evaluate whether a condition is present) |
| **Pt** | Patient |
| **Pt c/o** | Patient complains of |
| **w/o** | Without |
| **w/** | With |
| **↑** | Increased |
| **↓** | Decreased |
| **→** | Leading to / resulting in |
| **~** | Approximately |
| **q** | Every (from Latin *quaque*) — used in medication dosing |
