# World and Geography — Out-of-Character Glossary

Terms used by **players** when discussing the game world, map, and setting outside of character. Covers map knowledge, server world conventions, and how players navigate and reference the game environment.

---

## Map and Navigation

| Term | OOC Meaning |
|---|---|
| **The map** | The game world — Los Santos and Blaine County. The playable area of GTA V. |
| **Minimap** | The small radar in the bottom corner of the screen. Used for navigation — not always permitted as IC knowledge depending on server rules. |
| **Waypoint** | A GPS marker placed on the map. A player following a waypoint is navigating IC to a location. |
| **Fast travel** | A mechanic some servers offer to teleport to common locations instead of driving. Usually restricted to specific circumstances. |
| **Interior** | An accessible building interior in-game — apartment, office, shop, police station. |
| **MLO** | Map Location Object — custom-built interiors that replace or add to the default GTA V map. A server's MLO quality often defines the quality of location-based RP. |
| **Custom map / Custom build** | Server-added locations that do not exist in the base GTA V game. |
| **Locked out** | An area that is geographically present on the map but inaccessible due to collision or server restrictions. |
| **Out of bounds** | An area that players are not permitted to access — military zones, restricted areas, off-map regions. |

## World Conventions

| Term | OOC Meaning |
|---|---|
| **Server lore** | The accumulated history, events, and established facts of the server's version of Los Santos. May differ from GTA V canon. |
| **Canonically** | Recognized as officially having happened in the server's story. "That event is now canonical." |
| **Retcon** | Officially changing or erasing something previously established as server lore. Generally avoided. |
| **The server's LS** | Los Santos as it exists on this specific server — not exactly GTA V's map and not exactly real Los Santos. |
| **Persistent world** | The game world continues to exist and evolve even when individual players are offline. |
| **Server restart / Tsunami** | The regular server reset. Usually roleplayed as a natural event (earthquake, tsunami, etc.) that explains any physical resets. |
| **Greenzone** | A defined area on the map where hostile RP is prohibited — hospitals, police stations, spawn areas. |
| **Hotspot** | An area of the map where RP activity naturally concentrates — the pier, the strip, Mission Row. |

## Vehicles and Travel

| Term | OOC Meaning |
|---|---|
| **Spawn vehicle** | The in-game command or menu to bring a vehicle into the world. |
| **Garage** | In-game system for storing owned vehicles and retrieving them. |
| **Impound** | The system or location where vehicles seized by police or abandoned in restricted areas are stored. |
| **Car meet** | A player-organized gathering of vehicle enthusiasts IC — a cruise, a show and shine, or a race event. |
| **Civilian vehicle** | An ordinary non-emergency, non-military, non-criminal vehicle — a standard road car. |
| **Emergency vehicle** | Police car, ambulance, fire engine — vehicles with lights and siren capabilities. |
| **Helipad** | A designated landing area for helicopters. Helipads exist at hospitals, police stations, and several buildings. |
