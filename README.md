# GTA V Roleplay Guides and Handbooks

A **free, open, community-driven** collection of roleplay guides and handbooks for GTA V across all multiplayer platforms — FiveM, RageMP, alt:V, and beyond.

---

## Why This Exists

The project began with the initial idea of archiving many old roleplay documents (but too precious to delete). It first it was just a bunch of documents. However, it turns out that these documents still can be put into a good use.

There seem to be no single, centralised source of GTA V roleplay documentation. Guides are scattered across Discord servers, forum threads, and private Google Docs — often locked behind server memberships or deleted when a community folds. This repository exists to change that: a permanent, public, platform-agnostic archive that anyone can read, copy, improve, and build upon.

Disclaimer: This project was assisted by AI for more accurate contents. And the first author's native language is not English.

## How to Use the Guides

1. **Browse** — Navigate the folders above or use GitHub's search to find a topic.
2. **Download** — Click any file and use the **Raw** or **Download** button, or clone the entire repository:
   ```bash
   git clone https://github.com/Apollo-Flight-Program/GTA-V-Roleplay-Guides-and-Handbooks.git
   ```
3. **Share** — All content is released under CC0 (public domain). You are free to share, adapt, and redistribute it without restriction or attribution.

## How to Contribute

We welcome guides, handbooks, corrections, and improvements from anyone. Please read [CONTRIBUTING.md](CONTRIBUTING.md) for the full process, but the short version is:

1. Fork this repository.
2. Add or edit files in the appropriate folder.
3. Open a Pull Request describing what you added or changed.
